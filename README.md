# 📚 TLib — String Extensions Library

## 📖 Описание

**TLib** — это библиотека на C#, предоставляющая набор *методов расширения для работы со строками*.

Она фокусируется на:

* безопасной работе со строками
* удобном поиске и замене
* преобразовании форматов именования
* анализе и обработке текста

---

## ✨ Возможности

* 🔍 Расширенный поиск подстрок
* 🔄 Гибкая замена и удаление
* 🔤 Преобразование регистра и форматов (camelCase, snake_case и др.)
* 📏 Выравнивание текста
* 🧠 Анализ строк (палиндромы, подсчёты)

---

## 💻 Пример использования

```csharp
using TLib;

string text = "Hello World";

var sub = text.SafeSubstring(0, 5);
var replaced = text.ReplaceFirst("Hello", "Hi");
var camel = "hello world".ToCamelCase();
bool isPal = "level".IsPalindrome();
```

## 📊 Таблица методов

| Метод            | Описание                                                             |
| ---------------- | -------------------------------------------------------------------- |
| SafeSubstring    | Безопасно возвращает подстроку с защитой от выхода за границы        |
| IsNullOrBlank    | Проверяет, является ли строка null, пустой или состоящей из пробелов |
| IsNotBlank       | Проверяет, что строка содержит данные                                |
| IndexOfAny       | Находит первое вхождение любой из указанных подстрок                 |
| CountOccurrences | Подсчитывает количество вхождений подстроки (с опцией перекрытия)    |
| ReplaceFirst     | Заменяет первое вхождение подстроки                                  |
| ReplaceLast      | Заменяет последнее вхождение подстроки                               |
| Remove           | Удаляет все указанные подстроки из строки                            |
| RemoveWhere      | Удаляет символы по заданному условию                                 |
| ToCamelCase      | Преобразует строку в camelCase                                       |
| ToPascalCase     | Преобразует строку в PascalCase                                      |
| ToSnakeCase      | Преобразует строку в snake_case                                      |
| ToKebabCase      | Преобразует строку в kebab-case                                      |
| Capitalize       | Делает первый символ заглавным                                       |
| Uncapitalize     | Делает первый символ строчным                                        |
| Align            | Выравнивает строку (влево, вправо, по центру)                        |
| Reverse          | Переворачивает строку                                                |
| Repeat           | Повторяет строку заданное количество раз                             |
| IsPalindrome     | Проверяет, является ли строка палиндромом                            |
